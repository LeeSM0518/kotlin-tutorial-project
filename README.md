# POS Console Application

## 요구사항
* 시작
* 프로세스
* 종료

## UI / UX
1. 애플리케이션 시작
   ```
   ==== POS의 전원이 켜졌습니다 ====
   ```
2. 메뉴 출력
   ```
   =========== 메뉴 ============
   
   1. 재고 조회
   2. 상품 판매
   3. 종료
   
   =============================
   번호 입력 => 
   ```
   1. 입력된 번호가 존재하지 않으면 다시 입력할 수 있도록, 알림 메시지 출력 후 재입력
      ```
      번호 입력 => qwe
      **** 존재하지 않는 메뉴입니다 ****
      번호 입력 => 
      ```
3. 종료
   ```
   ==== POS의 전원을 끕니다. ====
   ```
